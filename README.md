Esse é um projeto web responsivo de uma capsula do tempo para exibir memórias em uma linha do tempo.

Esse projeto foi desenvoilvido durante a NLW da Rocketseat com as seguintes teclonogias:
 - HTML
 - CSS
 - Git e Github
